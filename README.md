## Build

```
git clone  https://github.com/crackozabl/python-project-lvl1/
cd python-project-lvl1
pip install poetry
make install
make build

```

## Install

```
make package-install
```


## Play

```

brain-even
brain-gcd
brain-prime
brain-progression
```
